Open Source Windows Event Collector.

OpenWEC allows you to collect logs from Windows machines to Linux Server using native Windows Event Forwarding and Event Collector. This makes it easy to collect logs over HTTP, providing flexibility and higher performance than other alternatives.

  • Github: Open WEC
  • Stack: Python FastAPI Docker-Compose Nginx MongoDB MQTT

Microservice Architecture with FastAPI and Docker.

MicroAPI is fully separates API in Async mode based on Microservices

For CRUD operations is used Customer - Product - Price - Order model
The Customer buys a Product with a different Price and receives an Order
The Price is calculated including taxes/discounts depending on the type of customer

Each part is work like Microservice
The Microservice runs in separate Docker container

  • Github: Micro API
  • Stack: Python FastAPI Docker-Compose Nginx Postgres

Reative Frontend and fast Backend.

ReactAPI is an example of user interface based on
JavaScript library React on the Frontend
FastAPI and Postgres on the Backend

For CRUD operations is used Tickets-Comments model
The user can create Tickects and add Comments to them

The UI is in reactive mode and notifies changes immediately
Backend works in async mode with high performance and lightweight API

  • Github: React API
  • Stack: Python FastAPI React Docker-Compose Postgres Nginx
